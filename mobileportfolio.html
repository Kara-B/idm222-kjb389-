Mobile Portfolio: A Brief Summary

For anyone working in UX/UI, it's important to have an up to date portfolio online that showcases what you're capable of, what you've done in the past, and where your interests lie

To represent myself, I took my first try at a design for a portfolio visible on mobile devices.

This was my first time building a website from scratch and styling it, an experience which has taught me methods to take and avoid in the future when coding.

In my design process, I:

  • Used Figma to create mid-fi wireframes
  • Learned how to apply best practices in using HTML
  • Learned how to apply best practices in using CSS
  • Experimented with content layout on a basic level
  • Established a professional identity to be built upon
